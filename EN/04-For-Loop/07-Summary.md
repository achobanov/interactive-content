## In this lesson you learned: 

- For loops
  
  * overview of what for loops contains

  * for loops repeat a certain block of code

```java live
for (int i = 1; i <= 10; i++) {
  System.out.println(i);
}
```

- For loops with step

``` java
for (int i = 10; i >= 1; i--) {
  System.out.println(i);
}
```

- Iterating over characters

```java live
for (char ch = 'a'; ch <= 'd'; ch++) {
  System.out.print(ch + " ");
}
```

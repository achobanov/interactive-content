```java
while (condition) {
  loop body;
}
```

- Care este diferența dintre buclele **While** și **For** 

- Operatorul **Break** 

```java
Scanner scanner = new Scanner(System.in);
while (true) {
  int number = Integer.parseInt(scanner.nextLine());
  
  if (number % 2 != 0) {
    break;
  }
}
```

# Recapitulare

În această lecție am învățat:

- Buclele for
  
   * Prezentare generală a conținutului buclelor for

   * Buclele for repetă un anumit bloc de cod

```java live
for (int i = 1; i <= 10; i++) {
  System.out.println(i);
}
```

- Buclele for cu pas

``` java
for (int i = 10; i >= 1; i--) {
  System.out.println(i);
}
```

- Prefixul și sufixul

```java live
for (char ch = 'a'; ch <= 'd'; ch++) {
  System.out.print(ch + " ");
}
```
